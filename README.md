# 🛒 BlinkIT-Sales-Analysis

## 📌 Project Overview

This project presents an end-to-end data analysis of BlinkIT grocery sales data using Microsoft Excel.  
The objective was to clean, transform, analyze, and visualize the dataset to generate meaningful business insights through an interactive dashboard.

The project covers the complete analytics workflow:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Pivot Table Analysis
- Interactive Dashboard Development
- Business Insight Generation

---

## 🎯 Business Objectives

- Analyze overall sales performance
- Compare sales across item types and outlet types
- Evaluate fat content distribution
- Track year-wise sales trends
- Measure average product ratings
- Build an interactive dashboard for decision-making

---

## 🗂 Dataset Description

The dataset contains 8.5k grocery sales data with the following attributes:

- Item Type
- Fat Content (Low Fat / Regular)
- Outlet Type
- Outlet Tier
- Year of Establishment
- Sales Amount
- Item Rating
- Number of Items Sold

## 📊 Key Insights

- Low Fat products contributed approximately 65% of total sales.
- Supermarket Type 1 generated the highest revenue (~65% share).
- Fruits & Vegetables and Snack Foods were top-performing categories.
- Sales showed consistent growth with a noticeable peak around 2018.
- Average product rating remained stable at approximately 3.97.

---
## 📈 Recommendations

- Increase inventory and promotional efforts for high-performing categories like Fruits & Vegetables and Snack Foods.
- Expand the Low Fat product range, as it contributes the majority share of total sales.
- Replicate the successful strategy of Supermarket Type 1 across lower-performing outlet types.
- Implement targeted promotions and localized pricing strategies for underperforming outlets.
- Introduce cross-selling or bundling strategies to increase average sales per transaction.
